Limits

Intuitive Meaning:

The exprerssion $\displaystyle\lim_{x\to a}f(x)= L$ roughly means:

  1. The input $x$ can be arbitrarly approximated to $a$ (bigger and smaller).
  2. If the input $x\approx a$, then the output $f(x)\approx L$.
    3.
  3. The error in $f(x)\approx L$ can be made as small as desired by making the error in $x\approx a$ small enough (not neccessarily zero).
    4.

As the input $x$ approaches $a$, the output $f(x)$ approaches $L$.
